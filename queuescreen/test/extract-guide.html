yt-dlp doesn't support RTM Klik Malaysia, so we need to extract the stream URL manually using browser Developer Tools. Don't worry - it's easy!
1

Open RTM TV2 Page

First, open the RTM TV2 live page in your browser:

https://rtmklik.rtm.gov.my/live/tv2
Open RTM TV2
2

Open Developer Tools

Press F12 or right-click and select "Inspect"

Keyboard shortcuts:

  • Windows/Linux: F12 or Ctrl + Shift + I
  • Mac: Cmd + Option + I
üí° DevTools will appear at the bottom or side of your browser
3

Go to Network Tab

Click on the "Network" tab in Developer Tools

Then:

  1. Clear existing requests (click the üö´ or clear icon)
  2. In the filter box, type: m3u8
üí° Tip: The filter helps show only HLS stream files (.m3u8)
4

Reload and Play

Press F5 or Ctrl + R to reload the page

Wait for the video to start playing...

üì∫ Video should auto-play after a few seconds
5

Find the .m3u8 File

Look in the Network tab for requests ending in .m3u8

Common names:

  • master.m3u8
  • playlist.m3u8
  • index.m3u8
  • chunklist.m3u8
üí° Tip: Look for the one with "200" status code (successful)
6

Copy the URL

Right-click on the .m3u8 request ‚Üí "Copy" ‚Üí "Copy URL"

The URL will look something like:

https://rtm-live.glueapi.io/smil:rtm2/playlist.m3u8
https://stream.example.com/live/tv2/index.m3u8
‚úÖ You now have the direct stream URL!

Alternative: JavaScript Console Method

If Network tab doesn't show the URL, try this in the Browser Console:

  1. Open RTM page and wait for video to load
  2. Press F12 ‚Üí Console tab
  3. Paste this code and press Enter:
// Get video source
let video = document.querySelector('video');
console.log('Video src:', video.src || video.currentSrc);

// Or find all sources
let sources = Array.from(document.querySelectorAll('source'));
sources.forEach(s => console.log('Source:', s.src));

Look for URLs containing .m3u8 in the console output.
