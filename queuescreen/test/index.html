üìã For Raspberry Pi Deployment

Method 1: Extract direct stream URL (Recommended - No Ads!)

# On Raspberry Pi, install yt-dlp
pip3 install yt-dlp

# Extract the direct stream URL
yt-dlp -g "https://rtmklik.rtm.gov.my/live/tv2"

# This will output an m3u8 URL like:
# https://rtm-live.glueapi.io/smil:rtm2/playlist.m3u8

# Play with omxplayer (no ads, lightweight)
omxplayer --live "URL_HERE"

# Or with VLC (more robust)
cvlc "URL_HERE" --fullscreen --loop

Method 2: Use Chromium with ad-blocking

# Launch fullscreen browser
chromium-browser --kiosk \
  --autoplay-policy=no-user-gesture-required \
  "https://rtmklik.rtm.gov.my/live/tv2"
